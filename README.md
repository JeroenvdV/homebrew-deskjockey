# homebrew-deskjockey

Homebrew tap for [Deskjockey](https://github.com/JeroenvdV/Deskjockey) — a menu bar app that saves and restores multi-monitor display arrangements on macOS.

## Install

```bash
brew tap JeroenvdV/deskjockey
brew install --cask deskjockey
```

## Uninstall

```bash
brew uninstall --cask deskjockey
```
